KEY FEATURES OF THIS GAME

WORD BANK
The game includes a word bank feature
where a list of words is stored
and used for the player to guess.
This adds variety to the game
as different words can be randomly selected for each playthrough.
The word bank is be populated from a file.

VISUAL FEEDBACK
The game provides visual feedback to the player through the graphical interface.
The gallows and hangman's body parts are progressively drawn as the player makes incorrect guesses,
giving a visual representation of the player's progress and the number of remaining attempts.
This visual feedback adds engagement and excitement to the game.

USER INPUT VALIDATION
The game includes user input validation to ensure that the player enters valid inputs.
It checks if the input is a single alphabetic character
and prompts the player to enter again if the input is invalid.
This feature prevents incorrect inputs from affecting the gameplay
and provides a smoother user experience.
